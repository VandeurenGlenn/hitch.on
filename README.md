Hitch.on
--------
> Electron desktop remote app

## Info
Electron desktop remote app.
## Setup

### Prerequisites
- Nodejs

### Getting Hitch.on
#### zip
download the [zip](https://github.com//VandeurenGlenn/hitch.on/archive/master.zip)

#### git
```sh
git clone https://github.com/VandeurenGlenn/hitch.on.git
```
#### Install dependencies
```sh
npm install
```

## Running Hitch.on
### Run
```sh
  npm run start-dev
```

### Start file watcher
```sh
  tend
```
