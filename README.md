Hitch.on
--------
> Electron desktop remote app

## Info
Electron desktop remote app.
## Setup

### Prerequisites
- Nodejs

#### Global dependencies
##### Tend (file watcher)
Install Tend
```sh
npm install tend -g
```

## Running Hitch.on
### Run
```sh
  npm run start-dev
```

### Start file watcher
```sh
  tend
```
